# Flashcard Application User Guide

## Getting Started

### Account Setup
1. Visit the application at your deployed URL
2. Click "Register" to create a new account
3. Fill in your email and password
4. Verify your email (if enabled)

### Dashboard Overview
- **Quick Actions**: Create decks, start studying
- **Recent Decks**: Access your latest flashcard decks
- **Study Stats**: View your learning progress
- **Achievements**: Track your milestones

## Creating and Managing Flashcards

### Create a New Deck
1. Click "Create Deck" button
2. Enter deck name and description
3. Choose privacy settings (public/private)
4. Add tags (optional)
5. Click "Save"

### Add Flashcards
1. Open a deck
2. Click "Add Card"
3. Enter front and back content
4. Add tags (optional)
5. Click "Save"

### Study Mode
1. Select a deck
2. Click "Start Study"
3. Review cards:
   - Click to flip
   - Rate your knowledge (1-5)
   - Track progress

## Features

### Spaced Repetition
- Cards appear based on your performance
- More difficult cards show up more frequently
- System adapts to your learning pace

### Progress Tracking
- Daily study streaks
- Performance statistics
- Achievement system
- Learning analytics

### Deck Management
- Edit deck details
- Share decks
- Import/Export
- Archive/Delete

## Tips for Effective Learning

1. **Regular Practice**
   - Study daily
   - Keep sessions short (15-30 minutes)
   - Use spaced repetition

2. **Card Creation**
   - Keep content concise
   - Use clear language
   - Add relevant examples

3. **Organization**
   - Use meaningful deck names
   - Add descriptive tags
   - Group related content

## Troubleshooting

### Common Issues
1. **Can't Login**
   - Check email/password
   - Clear browser cache
   - Reset password if needed

2. **Study Progress Not Saving**
   - Check internet connection
   - Refresh page
   - Contact support

3. **Cards Not Showing**
   - Check deck settings
   - Verify card creation
   - Clear browser cache
