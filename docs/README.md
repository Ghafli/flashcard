## Quick Start

1. **Installation**
```bash
git clone https://github.com/Ghafli/flashcard.git
cd flashcard
npm install
```

2. **Configuration**
Create `.env.local` with required environment variables.

3. **Development**
```bash
npm run dev
```

4. **Production Build**
```bash
npm run build
npm run start
```
